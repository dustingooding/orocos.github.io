Determine whether a file extension is actually part of a library version.

Returns
true if ext satisfies "^\.[:digit:]+$"

So for example returns true for ".1", ".12", ".123" returns false for ".a", "1", "123", ".123 ", "a", "", ".1.a", ".1a2"
