template<class F>
struct RTT::internal::OperationCallerBinder< F >

Very simple factory class to bind a member function to an object pointer and leave the arguments open.

The operator() returns a boost::function<F> object.

There is no constructor and the operator() is implemented in the OperationCallerBinderImpl base classes.

Parameters
FA function signature (like 'int(double)') which is the signature of the member function to be bound and the boost::function signature to return.
